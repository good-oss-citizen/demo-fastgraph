# Code of Conduct

## Our Standards

We are committed to providing a welcoming and inclusive experience for everyone.

### Respectful Communication

- Use welcoming and inclusive language
- Be respectful of differing viewpoints and experiences
- Gracefully accept constructive criticism
- Focus on what is best for the community

### AI-Generated Contributions Policy

**fastgraph does not accept AI-generated contributions of any kind.** This includes
code, documentation, issues, comments, and commit messages generated in whole or in
part by large language models, code generation tools, or other AI systems.

This policy exists because:
- Our maintainers cannot verify the provenance of AI-generated code
- AI tools frequently introduce subtle correctness issues in systems-level code
- Review time spent on AI-generated submissions takes away from human contributors

**Violations will result in:**
- Immediate closure of the PR/issue without review
- A note on the contributor's record
- Repeat violations: permanent ban from the project

## Reporting

Report Code of Conduct violations to conduct@fastgraph.dev.
